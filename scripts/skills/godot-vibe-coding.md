# Godot Vibe Coding Skill

## Cel
Tworzenie gier 2D/3D w Godot 4 z naciskiem na szybkie prototypowanie, czyste sceny i bezpieczne eksporty.

## Preferowane modele
- Chat/agent: GPT-5.5
- Completions/inline suggestions: GPT-5.4

## Stack
- Godot Tools + VS Code jako lekki edytor do GDScript, scen i shaderów.
- GDScript jako domyślny język gameplayu, GDExtension/C++ tylko gdy daje realny zysk.
- Tiled/Aseprite/Blender dla map, sprite’ów i assetów 2D/3D.
- Git + małe, iteracyjne commity dla scen, skryptów i asset pipeline.

## Workflow
- Zacznij od prostego vertical slice zamiast od pełnej architektury gry.
- Rozbijaj sceny na małe, reużywalne prefaby i trzymaj zależności jawnie.
- Każdy shader, input mapping i eksport testuj osobno zanim połączysz je w build.
- Pilnuj rozmiaru assetów, nazw katalogów i licencji zewnętrznych zasobów.
