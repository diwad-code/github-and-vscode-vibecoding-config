# Game Asset Pipeline Vibe Coding Skill

## Cel
Uspójnienie pracy nad assetami do gier: sprite’y, tilemapy, modele, animacje, eksporty i porządek licencyjny.

## Preferowane modele
- Chat/agent: GPT-5.5
- Completions/inline suggestions: GPT-5.4

## Stack
- Blender do modeli, rigów i prostych bake’ów.
- Tiled do tilemap i layoutów poziomów.
- Aseprite lub LibreSprite do pixel-artu i animacji 2D.
- Repo z katalogiem assets, źródłami eksportu i krótką dokumentacją licencji.

## Workflow
- Trzymaj pliki źródłowe i eksporty osobno, żeby nie mieszać assetów roboczych z runtime.
- Ustal stałe nazewnictwo dla sprite’ów, atlasów, materiałów i animacji.
- Przed importem do silnika sprawdź skalę, pivoty, FPS animacji i format kompresji.
- Każdy zewnętrzny asset zapisuj z linkiem do licencji i informacją, czy wolno go modyfikować.
